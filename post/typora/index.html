标题

用一到六个#作为一行的开头对应标题的六个等级（注意”#后面的一个空格）

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

引用

当输入>会生成一个引用块。

例如

>我能吞下玻璃而不伤身体

效果

我能吞下玻璃而不伤身体

列表

无序列表

输入* list item 1创建无序列表。

例如

* C
* Python
* Java

效果

  • C
  • Python
  • Java

有序列表

输入1. list item 1 创建有序序列

例如

1. C
2. Python
3. Java

效果

  1. C
  2. Python
  3. Java

任务列表

输入-[ ]或-[x]表示完成或未完成

例如

- [ ] 看书
- [x] 锻炼

效果

代码块

输入```并按下回车创建代码块。

在```之后添加语言的名称可以对代码中关键字进行高亮。

例如

​```ruby
require 'redcarpet'
markdown = Redcarpet.new("Hello World!")
puts markdown.to_html
​```

效果

require 'redcarpet'
markdown = Redcarpet.new("Hello World!")
puts markdown.to_html
